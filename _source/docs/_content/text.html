--- title: Text order: 03 ---

.text

.text helps you masquerade text elements as other elements. Perfect for those times where you need to maintain semantic structure but want the appearance to be something other than the default element style provides. Make a H2 look like a H3, etc. It’s also useful for applying a scale factor to components as well as art directed text.

--size modifiers

--size modifies the text size of elements, using the same custom properties that are directly used by text elements and headings. --xl and up also modify line heights, as you almost always want less leading with larger type.

text --size-xxs

text --size-xs

text --size-s

text --size-m

text --size-l

text --size-xl

text --size-xxl

text --size-xxxl

text --size-xxxxl

--weight modifiers

--weight modifiers control the weight of the text. Out of the can, these expect a variable font. Adjust to the weights provided by your custom font as need be.

text --weight-200

text --weight-300

text --weight-400

text --weight-500

text --weight-600

text --weight-700

text --weight-800

--align modifiers

Use --align modifiers to manipulate an element’s alignment.

text --align-left

text --align-center

text --align-right

--leading modifiers

Give an element more or less leading with the --align modifiers.
