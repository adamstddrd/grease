---
title: Animation
---

Grease includes a simple, tiny, composable animation system based on scroll-triggered and scroll-linked CSS animation:

1. Use `animate-self`, `animate-children`, and `animate-scroll` to trigger animations as elements enter and exit the viewport. The animation is set to `fade-in` by default.
2. Change animations by creating and applying new animation utilities. See `.fade-in` for reference.
3. Use the `delay` and `duration` utilities to further refine your animations.
